# Go Forge

A deep dive into Go and backend engineering.

This repository is my laboratory for understanding Go deeply.

I am not collecting tutorials.

I am building mental models.

## Learning Principles

- Understand before abstracting
- Write the code from scratch
- Break things intentionally
- Benchmark when performance matters
- Read the standard library
- Document mistakes
- Build production-oriented projects

## Core Focus

- Go fundamentals
- Data structures
- Error handling
- Interfaces and composition
- Concurrency
- Networking
- Go internals
- Backend engineering
- Performance
